# CommandSat Backend

## Installation

Install dependencies.

```bash
npm install
```

## Usage

Start the backend server

```bash
npm start
```

Navigate to [localhost](https://localhost:3000/)

## Environment Variables

To run this project, you will need to add your environment variables to a `.env` file.  
You can create this file by copying the provided `.env.example` file and fill in the following values:

- Your N2YO API key

**Note:** The `.env` file should not be commited to version control and is already included in the `.gitignore` file.
